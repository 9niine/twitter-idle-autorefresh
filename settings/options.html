As a privacy proponent myself, I would like to briefly state that the add-on will NEVER record any data whatsoever (e.g. browsing habits). It does however require the storage permission in order to save & read the above user settings. It will also only activate on URLs matching *://twitter.com/*.

The add-on is freely available on GitHub for full transparency.
